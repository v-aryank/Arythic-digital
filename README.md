# Arythic Digital Innovation Platform

A premium digital innovation platform combining cutting-edge technology with research-driven insights. This project showcases a modern, high-performance web interface built with React and Vite.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS, Vanilla CSS
- **UI Components**: shadcn/ui
- **Animations**: GSAP, CSS Animations
- **3D Graphics**: React Three Fiber

## Getting Started

### Prerequisites

- Node.js (Latest version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd arythic-digital-innovation-research
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Development

The project uses standard Vite scripts:

- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run preview`: Preview production build
- `npm run lint`: Run linting
